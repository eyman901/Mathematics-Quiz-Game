# Mathematics-Quiz-Game

**A simple console-based quiz game written in C++.**

## Features
- View questions.
- Play the quiz.
- Update or delete questions.
- Add new questions.

## How to Run
1. Download the `projecttt.cpp` file.
2. Open it in any C++ IDE (e.g., Dev-C++).
3. Compile and run the program.
